# Real-Time-Smart-parking-system-misalignment-detection-
IoT Smart Parking System featuring web-based slot booking (6AM-12PM). Visualizes 3 slots: Green (Free), Red (Full), Yellow (Misaligned). Uses sensors to detect improper parking and lane crossing, optimizing space for large vehicles like SUVs. Automatically fines users for misalignment to enforce discipline. Tech: Arduino, HTML, Sensors.
